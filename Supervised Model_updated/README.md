# Loan Approval Prediction System

## 🚀 Project Overview
This project predicts loan approval status using a Machine Learning model.
The model is deployed using FastAPI and integrated with a Streamlit frontend.

## 🧠 Tech Stack
- Python
- Scikit-learn
- FastAPI
- Streamlit
- Uvicorn

## 🏗 Architecture
Streamlit (Frontend) → FastAPI (Backend API) → ML Pipeline Model

## ▶️ How to Run

### 1️⃣ Install dependencies
pip install -r requirements.txt

### 2️⃣ Run FastAPI
uvicorn api:app --reload

### 3️⃣ Run Streamlit
streamlit run frontend.py

## 📊 Features
- Real-time prediction
- Pipeline-based preprocessing
- REST API deployment
- Interactive UI
